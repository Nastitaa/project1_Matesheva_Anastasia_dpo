# Лабиринт Сокровищ
Текстовая приключенческая игра, в которой вы исследуете таинственный лабиринт, полный загадок, ловушек и скрытых сокровищ. Ваша цель — найти и открыть легендарный сундук с сокровищами.

# Особенности
*   Исследование мира: Перемещайтесь между различными комнатами, каждая со своим уникальным описанием и атмосферой.
*   Решение загадок: Встречайте головоломки, которые потребуют от вас смекалки, чтобы продвинуться дальше и получить награды.
*   Система инвентаря: Собирайте и используйте предметы, чтобы взаимодействовать с миром и открывать новые пути.
*   Случайные события: Будьте готовы к неожиданным встречам и происшествиям, которые делают каждую игру уникальной.
*   Динамические ловушки: Неверные действия могут привести к потере предметов или другим неприятностям.

# Установка
1.  Убедитесь, что у вас установлен Python 3.8 или выше.
2.  Клонируйте репозиторий или скачайте исходный код.
3.  Установите зависимости с помощью `poetry`:

make install
    
# Запуск
Запустите игру с помощью следующей команды:

make project

# Просмотреть запись игрового цикла
